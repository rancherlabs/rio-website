Intro header goes here

Rio is a user oriented end-to-end container solution with a focus on keeping containers simple and combating the current trend of complexity. It's kept fun and simple through its familiar and opinionted user experience. Additionally, Rio is a "Cloud Native Container Distribution", meaning is includes built-in Cloud Native technologies such as Kubernetes, Istio, Containerd, etc. so that the user does not need to be an expert in installing, using, and maintaining these system.

Features

Simple

Opinionated

Simplicity can only be achieved with a focused point of view.

Fun

Bringing the fun back

Containers should be fun.

Portable

Run Anywhere

Each Rio cluster has the same functionality available to it. The differences between providers are only speed, reliability, and permission controls. Running in production is as simple as your laptop.

Secure

By default

Rio defaults to the best security settings and encryption available.

Production-Grade

Scales to 11

Running Rio should give you a production-worthy system and not need more bolt-on ops tools.

Cloud Native

Buzzword compatible

Rio will include all the the key cloud native technologies by default so you don't need to be an expert in the details.
